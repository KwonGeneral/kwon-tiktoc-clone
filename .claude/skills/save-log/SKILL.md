현재 세션의 대화 내용을 요약하여 `.docs/ai_logs/` 에 저장합니다.

## 절차

1. 현재 작업 중인 Phase를 확인하세요.
2. 이번 세션에서 나눈 대화를 다음 형식으로 요약하세요:

```markdown
# Phase {번호} - {Phase 이름} 대화 로그

- **날짜**: {오늘 날짜}
- **Phase**: {번호} - {이름}
- **브랜치**: {현재 브랜치}

## 주요 프롬프트 및 결과

### 1. {작업 설명}
- **프롬프트**: "{사용자가 지시한 내용 요약}"
- **결과**: {Claude가 수행한 결과 요약}
- **변경 파일**: {변경된 주요 파일 목록}

### 2. {작업 설명}
- **프롬프트**: "..."
- **결과**: ...

## 사용자가 직접 수행한 부분
- {설계 결정, 코드 리뷰, 수정 지시 등}

## AI가 수행한 부분
- {코드 생성, 파일 생성, 명령 실행 등}
```

3. `.docs/ai_logs/phase-{번호}-{날짜}.md` 파일로 저장하세요.
   - 같은 Phase에서 여러 세션이면: `phase-{번호}-{날짜}-{순번}.md`
4. 저장 완료 후 파일 경로를 사용자에게 알려주세요.

## 규칙
- 실제 대화 내용을 그대로 복사하지 말고 요약
- 프롬프트는 핵심만 간결하게
- 내부 전략(과제 원문, 비공개 문서 내용)은 포함하지 않음
- 사용자가 직접 한 것과 AI가 한 것을 명확히 구분
- 이 로그는 비공개(.docs/)이지만, 나중에 선별하여 README에 포함할 수 있음
