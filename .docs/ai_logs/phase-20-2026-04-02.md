# Phase 20: 프로필 강화 + 설정

- **날짜**: 2026-04-02
- **브랜치**: feature/phase-20-profile-settings

## 작업 내용

### 1. User Entity 확장
- `User` entity에 `bio` 필드 추가
- `UserModel`에 `bio` 필드 추가 + `fromEntity`/`toEntity` 매핑

### 2. LocalStorage 확장
- `LocalStorageRepository`에 닉네임, 자기소개, 알림 설정 메서드 추가
- `LocalStorageService`에 SharedPreferences 기반 구현

### 3. 프로필 Provider 리팩토링
- `currentUserProvider` — 로컬 저장된 닉네임/자기소개 반영
- `likedVideosProvider` — 좋아요한 영상 목록
- `bookmarkedVideosProvider` — 북마크한 영상 목록
- `myVideosProvider` — 내가 업로드한 영상 목록
- `ProfileEditNotifier` — 프로필 편집 저장
- `NotificationSettingNotifier` — 알림 설정 토글

### 4. 프로필 편집 페이지
- `ProfileEditPage` 생성 (닉네임 + 자기소개 160자 제한)
- 취소/저장 버튼, 로컬 저장 후 Provider invalidate
- ProfileHeader "편집" 버튼 → 편집 페이지 이동

### 5. 프로필 탭 3개 + 영상 그리드
- 기존 4탭 → 3탭 (내 영상, 북마크, 좋아요)
- `ProfileVideoGrid` 위젯 (3열 그리드, 썸네일+조회수)
- 각 탭 빈 상태 처리

### 6. 설정 페이지
- `SettingsPage` 생성 (알림 토글, 권한→앱 설정 이동, 앱 버전)
- `app_settings` 패키지 추가
- 메뉴(≡) 버튼 → 설정 페이지 이동

### 7. 라우팅
- `/profile/edit`, `/settings` 경로 추가
