# Feature 23: 프로필 이미지 변경 — AI 대화 로그

- **날짜**: 2026-04-03
- **Phase**: Feature 23
- **브랜치**: feature/phase-23-profile-image

## 작업 요약

API V4에서 추가된 프로필 이미지 업로드/조회 API를 연동하여 프로필 사진 변경 기능 구현.

## 주요 변경

### 패키지
- `image_picker: ^1.1.2` 추가

### Domain Layer
- `UserRepository`에 `getProfileImageUrl`, `uploadProfileImage` 메서드 추가
- `LocalStorageRepository`에 `getProfileImageUrl`, `saveProfileImageUrl` 추가
- `VideoRepository.uploadVideo`에 `avatarUrl` 파라미터 추가

### Data Layer
- `UserRepositoryImpl`에 HTTP 호출로 프로필 이미지 업로드/조회 구현
  - `GET /videos/profile-image/:userId` → URL 조회
  - `POST /videos/profile-image/upload` → multipart 업로드
- `LocalStorageService`에 프로필 이미지 URL 캐싱 구현
- `VideoRemoteDataSource.uploadVideo`에 `avatarUrl`, `userId` 필드 전달

### Presentation Layer
- `ProfileImageNotifier` — 프로필 이미지 업로드 상태 관리
- `profileImageProvider(userId)` — API에서 프로필 이미지 URL 조회
- `currentUserProvider` — 로컬 캐시된 프로필 이미지 URL 반영
- `ProfileHeader` — 아바타 탭 → 바텀시트(갤러리/카메라) → 업로드 → 즉시 반영
- `ProfileEditPage` — 프로필 사진 표시 + "사진 변경" 버튼 추가
- `UserProfilePage` — API에서 해당 유저 프로필 이미지 조회/표시
- `PublishProvider` — 영상 업로드 시 프로필 이미지 URL을 avatarUrl로 전송

## AI 활용 범위
- 전체 구현을 AI가 수행 (설계 → 코드 작성 → 정적 분석)
- 사용자가 계획 승인 후 진행

## 변경 파일 수
- 12개 파일 수정/생성
