# Feature 24: 게시물 이미지 (사진 촬영 + 업로드)

- **날짜**: 2026-04-03
- **브랜치**: feature/phase-24-post-image

## 작업 내용

### Domain Layer
- `PostImage` 엔티티 생성 (freezed): id, thumbUrl, fullUrl, caption, userId, username, nickname, avatarUrl, createdAt
- `PostImageRepository` 인터페이스: getPostImages(), uploadPostImage()

### Data Layer
- `PostImageModel` (freezed + json_serializable): API JSON 파싱 + toEntity() 변환
- `PostImageDataSource` 인터페이스 + `PostImageRemoteDataSource` 구현
  - GET /videos/images → 이미지 목록 조회 (캐싱)
  - POST /videos/images/upload → multipart 이미지 업로드
- `PostImageRepositoryImpl`: DataSource → Entity 변환 위임

### Presentation Layer
- **카메라 화면**: 동영상 | 사진 모드 전환 탭 추가, 사진 모드에서 흰색 원형 촬영 버튼
- **이미지 게시 화면** (`PublishImagePage`): 이미지 미리보기 + caption 입력 + 업로드
- **이미지 상세 화면** (`ImageDetailPage`): CachedNetworkImage + InteractiveViewer (확대/축소)
- **프로필 페이지**: 내 게시물 탭에 영상 + 이미지 혼합 그리드
- **유저 프로필 페이지**: 해당 유저 이미지 게시물도 표시
- `ProfileImageGrid` 위젯: CachedNetworkImage 썸네일 그리드 (정사각형, 이미지 아이콘 표시)

### 기타
- `cached_network_image` 패키지 추가
- GoRouter에 /publish-image, /image/:imageId 경로 추가
- DI providers에 PostImageDataSource, PostImageRepository 등록
