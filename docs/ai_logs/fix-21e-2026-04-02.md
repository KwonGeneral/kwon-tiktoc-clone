# Fix 21E: 팔로잉 + 게시 + 알림 + 비디오 안정성

- **날짜**: 2026-04-02
- **작업 시간**: ~1시간
- **AI 도구**: Claude Code (Opus 4.6)

## 작업 내용 (7개 버그 수정)

### Bug 1: 팔로잉 탭 빈 상태에서 TopTabBar 사라짐
- **원인**: TopTabBar가 VideoOverlay(VideoCard 내부)에만 있어서 displayVideos가 비면 렌더링 안 됨
- **수정**: `feed_page.dart`에서 빈 상태를 Stack으로 감싸 TopTabBar를 항상 표시

### Bug 2: 게시 버튼 시 키보드 올라옴
- **원인**: `FocusScope.of(context).unfocus()`가 불안정
- **수정**: `FocusManager.instance.primaryFocus?.unfocus()` + `_descriptionFocusNode.unfocus()` 명시적 호출

### Bug 3: 게시 완료 후 프로필 미이동
- **원인**: listener에서 즉시 navigation → 오버레이와 타이밍 충돌
- **수정**: 800ms 딜레이 후 `feedNotifier.reload()` → `context.go(profile)`, prev 상태 체크로 중복 호출 방지

### Bug 4: 앱 재실행 시 업로드 영상 미표시
- **원인**: `addUploadedVideo()`는 로컬 파일 경로로 메모리에만 추가
- **수정**: `feedNotifier.reload()` 메서드 추가, 업로드 후 API에서 서버 URL로 피드 리로드

### Bug 5: 프로필 탭 전환 시 그리드 미갱신
- **원인**: `_VideoThumbnail`의 `AutomaticKeepAliveClientMixin`으로 위젯 상태 재활용
- **수정**: `ProfileVideoGrid`에 `ValueKey('my_videos'/'bookmarks'/'likes')` 추가, 각 썸네일에 `ValueKey(video.id)` 추가

### Bug 6: 알림 설정 시스템 알림과 동기화
- **원인**: 토글이 로컬 상태만 저장, 시스템 알림 상태 무시
- **수정**: OFF→ON 전환 시 `AppSettings.openAppSettings(type: AppSettingsType.notification)` 호출

### Bug 7: 비디오 무한로딩 (MediaCodec dead thread)
- **원인**: `_initController` 비동기 실행 중 `_onPageChanged`에서 dispose → 레이스 컨디션
- **수정**: `_disposed` Set 추적, 초기화 후 dispose 여부 확인, 10초 타임아웃, try-catch로 dispose 실패 무시
