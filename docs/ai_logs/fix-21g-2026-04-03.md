## 작업 내용

### 1. 게시 버튼 시 키보드 안 올라오게
- `_startUpload()`에 `FocusManager.instance.primaryFocus?.unfocus()` 추가
- 다이얼로그 닫힌 후 키보드가 다시 올라오는 문제 해결

### 2. 게시 완료 후 프로필 내 영상 표시
- `PublishState`에 `uploadedVideoUrl`, `uploadedDescription` 필드 추가
- `publish()`에서 서버 반환 Video URL 저장
- `_onUploadSuccess()`에서 `addUploadedVideo()`로 피드에 즉시 추가 후 프로필 이동
- 비동기로 `reload()` 호출하여 서버 데이터도 갱신

### 3. 팔로잉 탭 필터링 + 탭바 유지
- 비디오 userId 할당을 `user_${v['id']}` → `user_${index + 1}`로 변경
- mock user(user_1~user_13)와 일관된 매핑 보장
- 빈 상태에서 TopTabBar가 이미 정상 렌더링되는 것 확인

### 4. 프로필 탭 전환 시 영상 그리드 갱신
- `ref.watch(feedNotifierProvider)` 기반 derived provider로 이미 정상 동작 확인
- 별도 수정 불필요

### 5. 알림 설정 시스템 동기화
- `permission_handler` 패키지 추가
- `NotificationSettingNotifier`에 `_syncWithSystem()`, `checkSystemStatus()` 메서드 추가
- `toggle()` 개선: 시스템 알림 OFF 시 설정 이동만 하고 토글 안 함
- `SettingsPage` → `ConsumerStatefulWidget` + `WidgetsBindingObserver`로 변경
- resume 시 시스템 알림 상태 재확인

### 6. 영상 업로드 시 모든 필드 전송
- `VideoRepository`, `VideoDataSource`, `VideoRemoteDataSource`, `VideoRepositoryImpl` 전체 레이어에 title, tags, location, username, nickname 파라미터 추가
- `PublishNotifier`에서 프로필 닉네임, 사용자 정보 포함하여 업로드

### 7. 상대경로 → 절대경로 변경
- Python 스크립트로 31개 파일의 상대 import를 `package:kwon_tiktoc_clone/...` 절대경로로 일괄 변환

### 8. Optimistic Update 일괄 적용
- 좋아요/북마크/팔로우 토글: 이미 적용됨 확인
- 댓글 로딩: 로컬 댓글을 먼저 즉시 표시 → 서버 댓글 비동기 병합으로 개선
- 댓글 작성: 이미 즉시 추가됨 확인

### 9. 댓글창 올라올 때 영상 축소
- `showModalBottomSheet` → 인라인 뷰(`CommentInlineView`) 전환
- `CommentState`에 `isOpen` 필드 추가
- `CommentNotifier`에 `open()`, `close()` 메서드 추가
- `VideoCard`에 `_buildWithComments()`: 영상 38% + 댓글 62% 레이아웃
- 닫기 버튼으로 원래 전체화면 영상으로 복귀
