# Phase 14: 댓글 기능 - AI 대화 로그

- **날짜**: 2026-04-02
- **Phase**: 14 - 댓글 기능
- **브랜치**: feature/phase-14-comment-feature

## 작업 내용

### 구현 항목
1. Mock 댓글 데이터 보강 (영상별 5~10개, 유저명/내용 다양화)
2. 댓글 바텀시트 UI (DraggableScrollableSheet, 헤더/목록/입력)
3. 댓글 목록 표시 (아바타, 이름, 내용, 시간, 좋아요 수)
4. 댓글 좋아요/싫어요 토글 (로컬 저장)
5. 댓글 입력 → 로컬 저장 → 목록 추가 → commentCount 업데이트
6. 1depth만 (답글 UI는 표시되지만 비기능)

### 새로 생성한 파일
- `lib/presentation/feed/provider/comment_provider.dart` — CommentNotifier
- `lib/presentation/feed/provider/comment_state.dart` — CommentState (freezed)
- `lib/presentation/feed/widget/comment_bottom_sheet.dart` — 댓글 바텀시트
- `lib/presentation/feed/widget/comment_item.dart` — 개별 댓글 위젯

### 수정한 파일

- `lib/data/datasource/video_remote_datasource.dart` — 동일
- `lib/domain/repository/local_storage_repository.dart` — 댓글 저장 메서드 추가

- `lib/presentation/feed/widget/video_overlay.dart` — 바텀시트 연결
- `lib/presentation/feed/provider/feed_provider.dart` — incrementCommentCount

### 아키텍처 결정
- CommentNotifier를 FeedNotifier와 분리하여 관심사 분리
- 댓글 좋아요/싫어요는 로컬 Set으로 관리 (서버 API 없으므로)
- 사용자 작성 댓글은 SharedPreferences에 JSON으로 영속화
- 댓글 추가 시 FeedState의 commentCount 동기 업데이트
