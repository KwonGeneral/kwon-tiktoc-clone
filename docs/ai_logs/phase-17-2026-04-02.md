# Phase 17 - 촬영 (카메라 + 15초 녹화)

**날짜**: 2026-04-02
**브랜치**: feature/phase-17-camera-recording

## 작업 내용

### 1. 패키지 + 플랫폼 권한
- `camera: ^0.11.1` 패키지 추가
- iOS Info.plist: NSCameraUsageDescription, NSMicrophoneUsageDescription 추가
- Android AndroidManifest.xml: CAMERA, RECORD_AUDIO 퍼미션 추가

### 2. Provider 구현
- `CameraState` (freezed): RecordingStatus(idle/recording/paused), elapsed, isFrontCamera, recordedFilePath
- `CameraProvider` (riverpod_generator): 100ms 단위 타이머, 녹화 시작/일시정지/재개/정지/리셋, 카메라 전환

### 3. CameraPage UI
- 전체 화면 카메라 프리뷰 (FittedBox.cover로 화면 꽉 채움)
- 상단: 닫기(X) 버튼 + 카메라 전환 버튼
- 하단: 녹화 컨트롤 (빨간 원형 버튼 + 프로그레스 링)
- 녹화 중: 타이머 표시 + 프로그레스 바 + 취소/완료 버튼
- 15초 도달 시 자동 정지 (ref.listen)
- 취소 시 "마지막 클립을 삭제할까요?" 다이얼로그
- WidgetsBindingObserver로 앱 라이프사이클 처리

### 4. 라우터 + 네비게이션
- RoutePaths.camera = '/camera' 추가
- /camera 라우트를 ShellRoute 밖에 배치 (하단 네비 없는 전체 화면)
- MainShell에서 + 버튼(index 2) 탭 시 context.push('/camera')
- 기존 discover 브랜치는 SizedBox.shrink()로 변경

### 5. 상수/문자열
- AppConstants.maxRecordingDuration = 15초
- AppStrings에 카메라 관련 문자열 12개 추가
