## 작업 요약

### 구현 항목

1. **Model 클래스 3개** (freezed + json_serializable)
   - `VideoModel` - Video Entity 1:1 매핑, toEntity()/fromEntity() 변환
   - `UserModel` - User Entity 1:1 매핑
   - `CommentModel` - Comment Entity 1:1 매핑

2. **DataSource 레이어**
   - `VideoDataSource` 추상 인터페이스 (getVideoFeed, toggleLike, toggleBookmark, getComments)
   - `MockVideoDataSource` 구현체
     - R2 CDN 비디오 URL 13개 (full-hd/1~13.mp4)
     - 한국어 설명, 음악명, 댓글 Mock 데이터
     - 페이지네이션, 좋아요/북마크 토글 로직

3. **Repository 구현체**
   - `VideoRepositoryImpl` - VideoDataSource에 위임, Model↔Entity 변환
   - `UserRepositoryImpl` - Mock 유저 13명 + 현재 유저 데이터

4. **Riverpod DI Provider** (`lib/core/di/providers.dart`)
   - `@riverpod` 어노테이션 사용
   - DataSource → Repository → UseCase 체인
   - Ref 타입 사용 (deprecated된 커스텀 Ref 대신)
