# Phase 6: 오버레이 UI - 인터랙션

- **날짜**: 2026-04-01
- **Phase**: 6 - 오버레이 인터랙션

## 작업 내용

### 1. FeedNotifier에 toggle 메서드 추가
- `toggleLike(videoId)`: 낙관적 업데이트로 즉시 UI 반영 후 서버 동기화
- `toggleBookmark(videoId)`: 동일한 낙관적 업데이트 패턴
- `toggleFollow(userId)`: FeedState의 followedUserIds Set으로 팔로우 상태 관리
- `isFollowing(userId)`: 팔로우 여부 조회 헬퍼

### 2. FeedState에 followedUserIds 필드 추가
- `Set<String> followedUserIds` 추가 (freezed 재생성)

### 3. SideActionBar 인터랙션 연결
- `onLikeTap`, `onBookmarkTap`, `onFollowTap` 콜백 추가
- `isFollowing` 상태에 따라 프로필 아바타 뱃지 토글 (+/체크)
- `_ActionItem`에 `GestureDetector` 래핑
- `_ProfileAvatar`에 팔로우 상태별 테두리 색상 + 뱃지 변경

### 4. VideoOverlay를 ConsumerWidget으로 변경
- FeedNotifier의 toggle 메서드를 SideActionBar 콜백으로 전달

### 5. 더블탭 좋아요 애니메이션 (LikeAnimation)
- `AnimationController` + `TweenSequence`로 스케일+페이드 애니메이션
- 0 → 1.2 → 1.0 스케일 + 페이드인/아웃 (800ms)
- VideoCard를 `ConsumerStatefulWidget`으로 변경
- `onDoubleTap` → 좋아요 안 된 상태면 좋아요 처리 + 항상 애니메이션 표시
