# Phase 7: 하단 네비게이션 — AI 대화 로그

- **날짜**: 2026-04-01
- **Phase**: 7 — 하단 네비게이션
- **소요**: 1 세션

## 작업 요약

### 구현 내용
1. **BottomNavBar 위젯** — 틱톡 스타일 커스텀 하단 네비게이션 바
   - 5개 탭: 홈, 친구, + (가운데), 알림, 프로필
   - 가운데 + 버튼: 핑크+시안 배경이 좌우로 살짝 비켜 보이는 틱톡 시그니처 스타일
   - `_NavItem` (아이콘+라벨), `_CreateButton` (그라데이션 + 버튼) 분리

2. **MainShell 전환** — `ShellRoute` → `StatefulShellRoute.indexedStack`
   - `StatefulNavigationShell` 을 받아 탭별 독립 상태 유지
   - `goBranch()`로 탭 전환

3. **GoRouter 라우트** — 5개 `StatefulShellBranch`
   - feed, friends, discover(create placeholder), notifications, profile

4. **Placeholder 페이지** — FriendsPage, NotificationsPage 신규 생성
   - ProfilePage도 Phase 8용 placeholder 위젯으로 교체

5. **상수 추가** — `RoutePaths.friends`, `AppStrings` 네비게이션 라벨

## AI 활용 방식
- 프로젝트 구조 탐색 → 기존 코드 패턴 파악 → 일관된 스타일로 구현
- StatefulShellRoute.indexedStack 패턴 적용 (탭 상태 보존)
